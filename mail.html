 2024. 02. Update Note
대시보드 기능
일정(Time) 분석 요약, 범위(Scope) 분석 요약, 자원(Resource) 분석 요약, 한눈에 제품(서비스)의 프로젝트 현황을 파악하며 관리 할 수 있습니다.
대시보드 바로가기
간트 차트 기능
전통적인 프로젝트 관리 방법과 최신화된 ARMS-ALM 연결을 통해 요구사항을 기존 방식대로 관리하면서 ARMS의 최신화된 기능을 활용 할 수 있습니다.
간트 차트 바로가기
일정 (Time) 분석 #1
제품(서비스)의 일정 진척율을 확인 할 수 있습니다. 100일의 프로젝트가 현재 50일이 지났다면 진척율은 몇일까요? 보고 받은 일정 진척율은 50% 입니까?
일정 분석 바로가기
일정 (Time) 분석 #2
요구사항이 언제 등록되고, 요구사항이 어떻게 처리되고 있는지, 지금 진척이 되고 있는지, 아니면 손 놓고 있는지, 관리자분들은 정확하게 알고 계십니까?
일정 분석 바로가기
범위(Scope) 분석 #1
제품(서비스)의 요구사항이 어떻게 분포하고 있는지 확인하고 계십니까? 요구사항이 어떤 이슈와 연결되어 있는지 확인하고 계십니까?
범위 분석 바로가기
범위(Scope) 분석 #2
버전 별 요구사항 개수, 각 요구사항의 상태, 요구사항에 어떤 사람들이 얼만큼의 이슈를 연결하고 있는지 분포를 확인 할 수 있습니다.
범위 분석 바로가기
자원(Resource) 분석 #1
버전 별로 어떤 사람들이 관여되 있는지. 각 사람들은 어떤 요구사항을 해결하고 있는지 개별적인 사람별로, 어떤 상태값을 가지고 있는지 확인할 수 있습니다.
자원 분석 바로가기
자원(Resource) 분석 #2
투입 인력별 요구사항의 총합을 대상으로 상태에 대한 통계와 투입 인력이 생성한 이슈의 상태 통계 및 퍼포먼스 통계에 대한 내용을 확인 할 수 있습니다.
자원 분석 바로가기
