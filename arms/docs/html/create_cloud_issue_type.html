클라우드 이슈 유형 생성

A-RMS에서 요구사항 이슈를 관리하는 데 필요한 arms-requirement 이슈 유형을 생성하는 과정

프로젝트의 관리 유형마다 이슈 유형 생성 방법에 차이가 있다.

O: 1, 2번 방법으로 이슈 유형 생성 가능

X: 1번 방법으로만 이슈 유형 생성 가능

6. 생성된 arms-requirement 이슈 유형 확인
